git:分布式版本控制软件

常用命令：

git add:将本地文件增加到暂存区

git commit:将暂存区的内容 提交到 本地仓库（本地分支，默认master分支）

git push:将本地仓库内容 推送到 远程仓库（远程分支）

git pull：将远程仓库（远程分支）内容 拉取到 本地仓库（本地分支）

git branch :查看所有分支

git checkout xxx:切换分支到xxx

git checkout xxx -f：强制切换分支到xxx

git branch xxx：创建分支

git checkout -b xxx：创建并切换到xxx分支

git branch -d xxx：删除xxx分支

git branch -D xxx：强制删除xxx分支

git tag v1.0 -a -m '这里是注释：表示添加名为v1.0的标签'

git show v1.0 ：表示展示出v1.0标签的全部内容

git tag :展示出所有标签

git push origin v1.0 :表示向远程推送名为v1.0的标签
